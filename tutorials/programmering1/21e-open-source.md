# Open source

Nu hvor du har en brugerkonto på GitHub har du mulighed for at bidrage til et open source projekter. Open source betyder at kildekoden til et projekt er offentligt tilgængeligt, og at alle har lov til at se, bruge, ændre og dele koden. Mange open source projekter hostes på GitHub, hvor du kan finde projekter indenfor næsten alle tænkelige områder.
1. Prøv at bruge søgefunktionen på GitHub til at finde et open source projekt der interesserer dig.
2. Måske kan du finde inspiration til Java kodning i andres projekter?

### Bidrag til et open source projekt
1. Følg instruktionerne i projektets README eller CONTRIBUTING.md.
2. Kig i projektets issues for at se om der er nogle opgaver du kan hjælpe med. Mange projekter har en "good first issue" label på issues der er velegnede for nye bidragydere.
3. Hvis du vil bidrage med en ændring til projektet, så kan du "forke" projektet til din egen GitHub konto, klone det ned til din computer, lave dine ændringer, lave et commit og pushe dine ændringer til din egen fork af projektet.
4. Når du har pushet dine ændringer til din egen fork, kan du lave en "pull request" til det oprindelige projekt, hvor du foreslår at dine ændringer bliver integreret i det oprindelige projekt.
5. Projektets vedligeholdere vil så gennemgå dine ændringer og beslutte om de vil acceptere dem eller ej.
6. Hvis dine ændringer bliver accepteret, så vil de blive integreret i det oprindelige projekt, og du vil have bidraget til et open source projekt!
