**Øvelse med boolean variable og udtryk**

Vi har disse variable med følgende værdier:

int x \= 27;

int y \= \-1;

int z \= 32;

boolean b \= false;

Hvad er værdien af følgende udtryk:

| !b |  |
| --- | --- |
| b \|\| true |  |
| (x \> y) \&\& (y \> z) |  |
| (x \=\= y) \|\| (x \<\= z) |  |
| ! (x % 2 \=\= 0\) |  |
| (x % 2 !\= 0\) \&\& b |  |
| b \&\& !b |  |
| b \|\| !b |  |
| (x \< y) \=\= b |  |
