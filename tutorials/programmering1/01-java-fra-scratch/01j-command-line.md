# Command Line Crash Course

Man kan sagtens skrive og køre Java programmer uden at bruge kommandolinen på sin computer, men det er en god idé at lære det alligevel. Det giver os en bedre forståelse af, hvad der sker "bag kulisserne" i IntelliJ IDEA.

Prøv at gennemføre så mange af øvelserne i [Command Line Crash Course](https://learnpythonthehardway.org/book/appendixa.html) som muligt. Forvilder du dig ind på sider, der handler om Python, så hop tilbage til den oprindelige side via linket ovenfor.
